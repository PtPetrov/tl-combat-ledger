# TL Combat Ledger

[Download the latest Windows build](https://ptpetrov.github.io/tl-combat-ledger/) (GitHub Pages landing)

> Note: This repository currently does not include a license. Treat the code and binaries as all rights reserved—do not redistribute or reuse without permission.

TL Combat Ledger is a Throne & Liberty combat log analyzer. It parses saved logs and visualizes rotations, skills, targets, DPS/HPS, crit rates, and timelines.

## Features

- Parse TL combat logs and surface per-target sessions, timelines, and breakdowns.
- Compare skills, damage, healing, crits, and rotation timing.

## Getting started (Windows)

1. Download the latest installer from the landing page above.
2. Run the setup (`TL-Combat-Ledger-<version>-setup-x64.exe`) and open the app.
3. Point the app at your TL combat log folder (or use the default path if detected).
4. Use “Check for updates” in the header to pull new releases when available.
